# 06 — Lessons (the graveyard)

The most valuable page. ~18 builds; their post-mortems are the real ROI. One line
each — brutally honest, scannable. **Almost all died the same death: no spread
loop.** That's why [03-distribution](03-distribution.md) is the spine and
`@jfun/growth-loop` is the hero.

## Games

| Project | What it was | Why it stalled |
|---|---|---|
| **Block Blossom** | Gen-1 endless block puzzle | No differentiation → felt flat. Gen-1 clone with nothing ownable doesn't work. |
| **Gravity Flip** | Flappy-style | **Apple-rejected as "design spam"** — cloned a specific iconic title. Genre mechanics OK, specific hits not ([01](01-apple-rules.md)). |
| **Tinker Lab** | Build/tinker toy | Published, no traction. **Cool ≠ comprehensible/marketable.** |
| **Underroot** | Complex systems game | Built on early rec — too complex. **Complexity kills; tutorial must be one sentence.** |
| **Vinegrow** | 3 species × growth × prune | MVP felt too complex. **One mechanic hid four systems** — hold a hard complexity budget. |
| **Thawline** | Melt + place/match | Graspability fail at self-test — two simultaneous systems, per-piece state. **Twist must be felt, not computed.** |
| **Moraine** | Solve-to-par gravity puzzle | Genuinely differentiated + solver-proven — **but shipped loop-less** (the old default). Fixed: adopted `@jfun/growth-loop` as v1.1 ([03](03-distribution.md)). |

## The one pattern

Every stall is some flavor of: **polished thing, no structural reason to spread or
return.** Differentiation + simplicity get you a *good* product; only the loop gets
you *distribution*. The studio's anti-patterns: don't out-content an incumbent
(Calm is declining on paid content) — out-*loop* them; don't build a solo utility
in a category that's emotionally huge but has no friend loop — that absence is the
opportunity, but only if you add the loop.
